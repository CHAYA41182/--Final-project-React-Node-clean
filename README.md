# התקנה
### דרישות מוקדמות: 
* node.js
* mongoDB compass
* mongoDB srver
  ##
  צריך ליצור אצלך במחשב מסד נתונים mongoo בשם  policmanDB
ואז לייבא אליו את חמשת הטבלאות שנמצאות בתיקייה בתוך תיקיית הפרוייקט.
צריך לשים לב שהם מתעדכנות מידי פעם ואז צריך לעדכן אותם גם אצלך.
## הפעלה
#### צד שרת:
```bash
cd server
npm i 
npm run dev
```
#### צד לקוח
```bash
cd client
npm i 
npm start
```
